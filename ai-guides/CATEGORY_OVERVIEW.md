# Category Overview

## Current Structure

**Exclusions Location**: Exclusions are currently embedded in the `description` field using "NOT:" statements. For example:
- `description: 'Medical expenses, pharmacy, health apps (NOT gym/fitness - use Sports category)'`
- `description: 'Movies, streaming, games, entertainment subscriptions. Examples: Netflix, Spotify, Disney+, HBO, gaming subscriptions. NOT: educational courses (use Education), gym/fitness (use Sports)'`
